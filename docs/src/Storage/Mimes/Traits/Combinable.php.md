# Combinable.php
<!-- DOCGEN:END -->

## Назначение

`trait Combinable` объединяет (конкатенирует) несколько файлов-ассетов одного типа в один результирующий файл, с кешированием результата на диске. Подмешивается в `CssMime`, `JsMime`, `SvgMime`, `FontMime`.

Сам трейт не знает, как именно склеивать содержимое разных типов файлов — он лишь организует поиск исходников, инвалидацию кеша по времени изменения и запись результата. Конкретный способ объединения (`__combine`) реализует класс, который подмешивает трейт.

## Публичные методы

### `combine($files, array $config = []): File`

Объединяет набор файлов в один и возвращает результирующий `File` (уже готовый, либо только что собранный и закешированный).

Параметры:
- `$files` — список файлов/URI для объединения. Разрешается через `File::find()` (если текущий файл — `is_uri`) либо через `$this->file->find()`; каждому элементу применяется fallback `'make'` (т.е. отсутствующий элемент будет создан как новый `File`, а не выбросит ошибку на этапе поиска).
- `$config` — опции:
  - `force` (bool) — принудительно пересобрать результат, даже если кеш свежий.

Поведение:
1. Разрешает `$files` в список объектов `File`; каждый URI-элемент подгружается через `fetch()`.
2. Если какой-то файл не существует на диске — бросает `\InvalidArgumentException` с сообщением `"File not found: {путь}"`.
3. Строит ключ кеша как `md5()` от списка путей файлов (порядок важен) и расширение результата берёт из `__combineExtension()` (по умолчанию — расширение текущего файла).
4. Кеш хранится через `CacheManager` в директории текущего файла, имя кеш-файла — `{ключ}.combined.{расширение}`.
5. Пересборка происходит, если передан `force => true`, либо если время изменения кеш-файла (`filemtime`) старше самого свежего `mtime` среди исходных файлов.
6. Возвращает результирующий объект через `File::make($cache->file)`.

## Расширяемость

Классы, использующие трейт, обязаны реализовать:
- `abstract protected function __combine(array $files, array $config): string` — собственно логика склейки содержимого (например, конкатенация с разделителями, обёртки и т.п.), должна вернуть итоговое содержимое результирующего файла.

Также можно переопределить:
- `protected function __combineExtension(): string` — расширение результирующего файла (по умолчанию — расширение исходного `$this->file`).

## Пример использования

```php
// $css — экземпляр CssMime
$bundle = $css->combine([
    'assets/reset.css',
    'assets/base.css',
    'assets/theme.css',
], ['force' => false]);

echo $bundle->getPathname(); // путь к объединённому и закешированному .css
```
