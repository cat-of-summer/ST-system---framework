# Minifiable.php

## 1. Концепция

Трейт для MIME-сервисов, которые умеют минифицировать контент (CSS, JS). Хранит минифицированный файл в кэше с TTL=-1 (навсегда). Повторный вызов `minify()` возвращает кэш без пересчёта, если исходный файл не изменился.

```php
// Используется через File::make() -> MIME-сервис
$css = File::make('~/assets/app.css');
$minified = $css->minify();       // CssMime::minify()
$minified = $css->minify(['force' => true]); // принудительная перегенерация
```

## 2. Публичные методы

### `minify(array $config = []): File`
Возвращает `File` на минифицированный файл (`*.min.ext`). Если кэш актуален и `force !== true` — возвращает кэшированную версию без пересчёта.

### `abstract __minify(string $content, array $config): string`
Должен быть реализован в использующем классе. Принимает сырой контент, возвращает минифицированный.
