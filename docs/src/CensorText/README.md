# CensorText

## Файлы

- [CensorText.php](CensorText.php.md)

<!-- DOCGEN:END -->

Модуль состоит из единственного самодостаточного класса `CensorText` — фильтра нецензурной/нежелательной лексики по пользовательскому словарю с группами слов. Умеет проверять текст на наличие плохих слов (`check`/`checkAll`), считать количество совпадений (`count`/`countAll`) и цензурировать текст, заменяя найденные слова на `*` (`censor`). Перед сопоставлением текст нормализуется (нижний регистр + опциональная замена похожих leetspeak-символов через настраиваемую `normalization_map`), а результаты кешируются в памяти инстанса по MD5 исходного текста.
