# HasHTMLRules

> Документация сгенерирована AI-агентом на основе исходного кода.

## 1. Концепция

Трейт для ботов и драйверов мессенджеров. Унифицирует преобразование HTML в нативный формат конкретного мессенджера. Используется в `TelegramBot`, `MaxBot` и подобных драйверах.

Класс, использующий трейт, должен реализовать абстрактный `getHtmlRules()` с маппингом HTML-тегов в нативные теги мессенджера.

## 2. Публичные методы

### `abstract protected static function getHtmlRules(): array`
Tаблица правил преобразования. HTML-тег → целевой тег или `false` (игнорировать) или `null` (развернуть).

### `normalizeHtml(string|\DOMDocument $html): array`
Преобразует HTML в массив нод. Строка автоматически парсится через `html_to_DOM()`.

### `normalize_url(string $url): string`
Нормализует URL: добавляет схему `https://` при её отсутствии..php
