Интеграции со сторонними сервисами и внутренняя маршрутизация:

- **`IntegrationDriver`** — абстрактный базовый класс для ВСЕХ драйверов в `Drivers/**`: декларативная регистрация методов-эндпоинтов (`registerMethod`), HTTP-транспорт через `HTTP\WebClient`, валидация параметров через `Rule`, кеширование ответов через `Cache\CacheManager`, событийный пайплайн на каждой стадии запроса. См. подробную документацию в `IntegrationDriver.php.md`.
- **`Router`** — маршрутизация входящих запросов приложения (в отличие от `IntegrationDriver`, который обслуживает исходящие вызовы к третьим сторонам).

`Drivers/` содержит конкретные интеграции, сгруппированные по категориям: платежи (`Acquiring/`), CRM (`CRM/`), гео-IP (`Geo/`), парсеры сайтов (`Parsers/`), мессенджер-боты (`Bots/`), AI-провайдеры (`AI/`), плюс несколько драйверов общего назначения прямо в `Drivers/` (капча, SMS, доставка и т.п.).
