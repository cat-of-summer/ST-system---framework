# Console

## Файлы

- [Command.php](Command.php.md)
- [Kernel.php](Kernel.php.md)

<!-- DOCGEN:END -->

Модуль реализует консольную подсистему фреймворка из двух связанных классов. `Command` — абстрактный базовый класс для конкретных CLI-команд: подкласс объявляет декларативную сигнатуру аргументов/опций в статическом свойстве `$signature` и переопределяет `handle()` с бизнес-логикой, а сам `Command` берёт на себя парсинг сигнатуры и резолвинг переданных аргументов/опций (доступны через защищённые `argument()`/`option()`). `Kernel` — статическое ядро, которое обнаруживает такие подклассы `Command` в настроенном каталоге (`registerDir()`), хранит их в реестре `сигнатура => класс` (`register()`) и диспатчит запуск из `$argv` (`handleCLI()`): разбирает командную строку на позиционные аргументы и опции, находит нужный класс по имени команды и создаёт его экземпляр, вызывая `handle()`.
