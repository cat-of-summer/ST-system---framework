# SmsRu

> Документация сгенерирована AI-агентом на основе исходного кода.

## 1. Концепция

Драйвер [sms.ru](https://sms.ru) API для отправки SMS-сообщений. `api_id` передаётся при инициализации и автоматически подставляется в каждый запрос.

```php
$sms = SmsRu::create('ваш-api-id');
$result = $sms->call('send', [
    'to'  => ['+79001234567', '+79007654321'],
    'msg' => 'Ваш код: 1234',
]);
```

## 2. Публичные методы

### `static create(string $api_id): static`

### `call('send', array $params): mixed`

| Параметр | Описание |
|---|---|
| `to` | Массив номеров телефонов (обязательно) |
| `msg` | Текст сообщения (обязательно) |
| `json` | Вернуть ответ в JSON-формате (опционально) |.php
