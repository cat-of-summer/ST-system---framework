# Mime.php

> Документация сгенерирована AI-агентом на основе исходного кода.

## 1. Концепция

Базовый класс MIME-сервиса. Все наследники получают `$this->file` (объект `File`). Автоматически выбирается в `File::make()` по MIME-типу.

## 2. Публичные методы

### `get(mixed $data): mixed`
Преобразует данные при чтении из кэша (базово — тъ же возвращает данные).

### `set(mixed $data, int &$flags = 0): mixed`
Преобразует данные при записи в кэш (базово — прозрачно).

### `toHTML(array $config = []): string`
HTML-представление. Базово — пустая строка.

### `static getAttrString(array $attrs): string`
Сериализует массив атрибутов в HTML-строку: `['class' => 'a', 'hidden' => true] -> 'class="a" hidden'`.
