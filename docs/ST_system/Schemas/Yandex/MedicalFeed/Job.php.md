# Job.php

`ST_system\Schemas\Yandex\MedicalFeed\Job` — XML-элемент `<job>` для Яндекс.Здоровье. Используется через поле `job` схемы `Doctor`.

## Поля

| Поле | Обяз. | Тип | Описание |
|------|-------|-----|----------|
| `organization` | да | string | Название организации |
| `period_years` | нет | string | Период работы (`'2010-2015'`, `'2018-н.в.'`) |
| `position` | нет | string | Должность |

## Использование

```php
'job' => [
    [
        'organization' => 'Городская клиническая больница №1',
        'period_years' => '2007-2015',
        'position'     => 'Врач-кардиолог',
    ],
    [
        'organization' => 'Клиника Здоровье',
        'period_years' => '2015-н.в.',
        'position'     => 'Заведующий кардиологическим отделением',
    ],
],
```
